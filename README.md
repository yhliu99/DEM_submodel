# MFDES

Code and examples for **MFDES** (Multi-Frequency Deep Energy Submodeling), a mesh-based deep energy framework for solid mechanics with localized high-gradient responses. The method combines mesh-based energy integration, a mask-DOF boundary treatment, and a two-stage global-local submodeling strategy with multi-frequency Fourier networks. fileciteturn2file0
